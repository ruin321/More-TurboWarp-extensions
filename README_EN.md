## üöÄ How to Use

### Using in TurboWarp

1. Open [TurboWarp](https://turbowarp.org/)
2. Click the "Extensions" button
3. Select "Custom Extension"
4. Paste the extension file URL or directly paste the code
5. Click "Load" to start using

### Using in Scratch

1. Download the extension `.js` file
2. In Scratch editor, click "Add Extension"
3. Select "Load extension from your computer"
4. Choose the downloaded `.js` file
5. The extension will appear in the blocks palette
