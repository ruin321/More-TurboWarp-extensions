## 🚀 使用方法

### 在 TurboWarp 中使用

1. 打开 [TurboWarp](https://turbowarp.org/)
2. 点击左下角"扩展"按钮
3. 选择"自定义扩展"
4. 粘贴扩展文件的 URL 或直接粘贴代码
5. 点击"加载"即可使用

### 在 Scratch 中使用

1. 下载扩展的 `.js` 文件
2. 在 Scratch 编辑器中点击"添加扩展"
3. 选择"从计算机中加载扩展"
4. 选择下载的 `.js` 文件
5. 扩展将出现在积木区中
